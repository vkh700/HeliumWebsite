Running from source

To run the web browser from source please download all of the browser requirements:

PyQt5
sys
PyQtWebEngine

Then obtain the source code using the git command:

cd ~/Desktop
git clone https://github.com/vkh700/Helium

After downloading the source you can run Helium by using python3 command:

python3 browser.py
